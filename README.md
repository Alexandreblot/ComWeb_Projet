Le but de ce projet est de créer un site web ecommerce fonctionnelle avec le web


Se connecter au local host --> ctrl-shift-p pour ouvrir les ficher sur debian, puis "php -S localhost:8000" dans la console pour lancer le serveur local, ensuite aller sur le navigateur et taper "localhost:8000" pour voir le projet.
